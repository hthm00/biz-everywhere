# biz-everywhere

Backend instructions

1. Install npm : [https://docs.npmjs.com/downloading-and-installing-node-js-and-npm]()
2. cd to the project in your terminal
3. Type in terminal

   ```
    npm i
   ```
4. Run the server (By default will run in [http://localhost:3000]())

   ```
   node index.js
   ```
